# Cómo Contribuir al Directorio

## Paso 1: Fork
Click en "Fork" arriba a la derecha

## Paso 2: Clone
```bash
git clone https://github.com/TU_USUARIO/TC2004B-S1-2026-ClassDirectory.git
cd TC2004B-S1-2026-ClassDirectory
```

## Paso 3: Crea tu Branch
```bash
git checkout -b add-TU_NOMBRE
```

## Paso 4: Crea tu Perfil
Crea `profiles/tu_nombre.md` usando `ejemplo_mauricio.md` como template

## Paso 5: Actualiza README
Agrega tu enlace a la lista en README.md

## Paso 6: Commit
```bash
git add .
git commit -m "Add [Tu Nombre] to class directory"
```

## Paso 7: Push
```bash
git push origin add-TU_NOMBRE
```

## Paso 8: Pull Request
Ve a GitHub y abre un Pull Request

## Paso 9: Code Review
Revisa los PRs de al menos 2 compañeros antes de que se acepte el tuyo
